#   Mi Blog 

##  El paso a paso

### 1. Cimientos del proyecto
Empecé creando la estructura :  `index.html` para el contenido, `style.css`, y un `script.js` . 

### 2. Control de versiones con Git
Aquí es donde inicialicé Git para rastrear cada cambio.

### 3. (GitHub)
Subí el proyecto a GitHub. Tuve que decidir entre usar **HTTPS**  o **SSH**. Al final, vinculé mi repositorio local con el remoto de forma exitosa.

### 4. Publicación 
El paso final fue activar **GitHub Pages**. Ahora, cualquier persona con el link puede ver mi trabajo en vivo.
